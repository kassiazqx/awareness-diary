# AI Start Here

这是 AI 协作者进入本项目时的冷启动入口。

## 项目一句话

觉察日记是一个本地优先的结构化觉察系统：用户记录日常事件、情绪、行动、念头，系统通过主线、标签、Todo、AI 分析帮助用户看见长期模式。

## 必读顺序

1. 先读 `docs/01-architecture/README.md`
2. 如果要理解产品，读 `docs/00-overview/product-concept.md`
3. 如果要看数据模型，读 `docs/00-overview/schema-visual.html` 和 `docs/03-schema/schema-register.md`
4. 如果要开发功能，读对应 `docs/02-design/<feature>/` 下的 spec / plan / decisions
